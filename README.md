# CRYPTOGRAPHIE – CRYPTANALYSE STATISTIQUE

## Description
Ce projet implémente un chiffrement mono-alphabétique ainsi qu’une méthode de cryptanalyse basée sur l’analyse de fréquence des lettres.  
L’objectif est de retrouver un texte en clair à partir d’un message chiffré.

## Fonctionnalités
- Implémentation du chiffrement mono-alphabétique
- Analyse de fréquence des lettres
- Aide à la décryption du message
- Validation avec des tests automatisés

## Structure du projet
- `cryptanalyse_subst.py` : script principal de cryptanalyse
- `frequence.py` : calcul des fréquences
- `subst_mono.py` : chiffrement / déchiffrement
- `cesar.py` : chiffrement de César (optionnel)
- `test_*.py` : tests

## Technologies
- Python

## Objectif
Comprendre les bases de la cryptographie classique et de la cryptanalyse statistique.
